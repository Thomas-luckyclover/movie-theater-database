# movie-theater-database
This project is a relational database system for managing a movie theater. It handles movies, screenings, customers, and ticket bookings. The goal is to model real-world relationships using SQL with proper normalization and constraints.

Overview

This project is a relational database system designed to manage a movie theater.
It handles movies, screenings, customers, and ticket bookings.

The database is designed using normalization principles to ensure data integrity and reduce redundancy.

Features

Relational schema with primary and foreign keys

One-to-many relationships (movies → screenings, customers → bookings)

Sample queries with JOIN and GROUP BY
